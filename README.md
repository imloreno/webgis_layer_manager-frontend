## üöÄ Installation

1. Clone the repository (careful, it's addictive):

```bash
git clone https://github.com/imloreno/webgis_layer_manager-frontend.git
```

2. Install dependencies with pnpm:

```bash
pnpm install
```

3. Start the development server:

```bash
pnpm run dev
```

4. For production builds:

```bash
pnpm run build
```
